#Sistema Bibliotecário

Primeiro projeto em C# e uma das formas que encontrei para começar a praticar os fundamentos de Programação Orientada a Objetos.

A ideia foi criar um sistema bibliotecário simples em console, onde fosse possível cadastrar livros e clientes, além de realizar empréstimos entre eles. O foco do projeto não é ser algo complexo, mas sim entender melhor como modelar entidades e trabalhar com objetos na prática.

## Funcionalidades

* Cadastro de livros
* Listagem de livros
* Cadastro de clientes
* Listagem de clientes
* Realização de empréstimos
* Consulta de empréstimos
* Encerramento do sistema

## O que pratiquei durante o desenvolvimento

* Programação Orientada a Objetos
* Classes e objetos
* Relacionamentos entre entidades
* Listas (`List<T>`)
* Estruturas de repetição
* `switch case`
* Aplicações de console
